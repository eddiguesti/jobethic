# 08 — Checklist Phase 0 — 30 jours

> À cocher au fil de l'eau. Si tu suis ça, tu auras 5 missions lancées à J+30.

---

## 🗓 Semaine 1 — Setup

### Lundi — Setup
- [ ] Créer le Notion workspace (cf [05-notion-crm.md](05-notion-crm.md))
- [ ] Créer les 5 bases (Freelances / Recruteurs / Missions / Rapports / Activity Log)
- [ ] Créer le dashboard d'accueil
- [ ] Setup Calendly avec 2 slots (15 min freelance / 30 min recruteur)
- [ ] Brancher Calendly à ton Google Calendar
- [ ] Créer le Google Form freelance (cf [03-form-freelance.md](03-form-freelance.md))
- [ ] Créer le Google Form recruteur (cf [04-form-recruteur.md](04-form-recruteur.md))
- [ ] Tester les deux Forms (les remplir toi-même)

### Mardi — Prospection freelances (4h)
- [ ] LinkedIn Sales Navigator ou recherche : sourcer 50 freelances ops/VA FR
- [ ] Envoyer 50 messages M1 freelance (cf [01-prospection-freelances.md](01-prospection-freelances.md))
- [ ] Logger les 50 dans Notion (Statut = "M1 envoyé")

### Mercredi — Prospection PME (3h)
- [ ] Sourcer 25 PME 11-50 salariés FR
- [ ] Envoyer 25 messages M1 recruteur (cf [02-prospection-pme.md](02-prospection-pme.md))
- [ ] Logger les 25 dans Notion

### Jeudi — Calls freelances qualifiés (2h)
- [ ] Trier les réponses freelances M1
- [ ] Booker 3-5 calls
- [ ] Mener les calls (30 min chacun, en visio, enregistrés avec accord)
- [ ] Envoyer M2 + Form aux personnes qualifiées
- [ ] Logger dans Notion

### Vendredi — Calls PME qualifiées (2h)
- [ ] Trier les réponses PME M1
- [ ] Booker 2-3 calls
- [ ] Mener les calls (30 min, en visio)
- [ ] Envoyer M2 + Form aux PME qualifiées
- [ ] Logger dans Notion
- [ ] **Bilan semaine** : "Cette semaine j'ai appris…" (Activity Log)

  - [ ] Rédaction du rapport (cf [07-template-rapport-match.md](07-template-rapport-match.md))
- [ ] Envoi du rapport à la PME

### Vendredi — Itération + 2ème matching (4h)

## ⚠️ Signaux d'alerte (à ne pas ignorer)

🚨 **Si < 5 % de réponse outbound** après 100 messages bien ciblés → le messaging est mauvais, raffine
🚨 **Si < 30 % des qualifiés remplissent le Form** → le Form est trop long ou pas clair, simplifie
🚨 **Si < 20 % des matchs aboutissent à un contact** → ton scoring est mauvais, revois la formule
🚨 **Si < 50 % des contacts aboutissent à une mission** → le profilage employeur est trop faible
🚨 **Si > 25 % de churn freelance à J+30** → on a mal compris les limites freelance
