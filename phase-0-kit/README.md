## Workflow Phase 0 — 30 jours

```
SEMAINE 1 — Setup + premières prospections
├── Lundi      : Setup Notion CRM (01)
├── Mardi      : 50 messages freelances LinkedIn
├── Mercredi   : 25 messages PME LinkedIn
├── Jeudi      : Calls qualifiés freelances (3-5)
└── Vendredi   : Calls qualifiés PME (2-3)

SEMAINE 2 — Premiers profils + missions
├── Lundi-Mardi : Profils freelances complétés (Forms)
├── Mercredi    : Profils PME complétés (Forms)
├── Jeudi       : Premier matching à la main (1-2 missions)
└── Vendredi    : Premiers rapports envoyés

SEMAINE 3 — Premières missions lancées
├── Lundi       : Lancement des 2-3 premières missions
├── Mardi-Vendr.: Continue prospections (50/sem) + suivi missions

SEMAINE 4 — Analyse + itération
├── Lundi       : Analyse des 5 premiers matchs
├── Mardi-Mercr.: Ajuste le scoring + questions
├── Jeudi-Vendr.: Démarrage Phase 1 (build no-code)
```

---

## KPIs Phase 0 (objectif Mois 1)

- 20 freelances qualifiés
- 10 PME engagées
- 5 missions lancées
- 3 missions arrivées à terme sans rupture
- NPS qualitatif > 50 sur 8+ conversations
