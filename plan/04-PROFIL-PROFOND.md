# 04 — Le Profil Profond : "Le Voyage"

> *"Le profil profond, c'est notre vrai produit. L'IA, le matching, l'app — tout le reste n'est que conséquence."*

## 1. Philosophie

Le profil profond n'est ni :
- ❌ un test de personnalité one-shot
- ❌ un questionnaire psychométrique
- ❌ une série de mini-jeux cognitifs
- ❌ un CV "enrichi" avec des soft skills

C'est :
- ✅ un **voyage de découverte de soi professionnelle**, vécu sur **plusieurs semaines**
- ✅ une **expérience** (pas une évaluation)
- ✅ une donnée qui **vit, mûrit, se met à jour** avec l'expérience professionnelle réelle
- ✅ un patrimoine que le candidat **possède** et qu'il peut emporter

**Référence d'expérience utilisateur :** Petit Bambou pour la méditation. Headspace. Calm. Reflectly.
**Référence de mécanique de profilage :** combinaison originale entre psychométrie Big Five revisitée, mises en situation Vervoe, et longitudinal-design.

---

## 2. La signature : "Le Voyage" en 3 actes

Une combinaison originale qu'aucun concurrent ne propose. Le tempo est lent et premium — l'antithèse des tests gamifiés bruyants américains qui choquent le marché FR.

### Vue d'ensemble

```
┌──────────────────────────────────────────────────────────────┐
│                         LE VOYAGE                            │
├──────────────────────────────────────────────────────────────┤
│                                                              │
│  ACTE 1 — LA DÉCOUVERTE              Jours 1-7   5 min/jour │
│  Style: Petit Bambou (calme, voix, thématique)               │
│  Capture: rythme, communication, autonomie, limites,         │
│           environnement, pression                            │
│                                                              │
│  ACTE 2 — LES MISES EN SITUATION     Jours 8-21  5 min/jour │
│  Style: Vervoe (simulations, scénarios branchants)           │
│  Capture: décision, conflit, négociation, stress, valeurs    │
│                                                              │
│  ACTE 3 — LES ÉCHOS                  Jours 22+   2 min/jour │
│  Style: Duolingo (mais ton FR-calme, streaks doux)           │
│  Capture: cohérence longitudinale, évolution, retour expé    │
│                                                              │
└──────────────────────────────────────────────────────────────┘
```

### Pourquoi 6 semaines minimum
- **Anti-triche structurel** : impossible de jouer un rôle pendant 30+ micro-sessions
- **Cohérence longitudinale** : les vraies préférences se révèlent quand on n'est plus en mode "premier jour"
- **Engagement** : on lit le candidat en plusieurs états émotionnels (lundi matin, vendredi soir, lendemain de mission)
- **Compounding** : chaque session améliore la précision, le candidat ne veut pas recommencer ailleurs

### Mais on n'attend pas 6 semaines pour proposer des missions
- **Pré-profil disponible dès J+3** (Acte 1 mid-way) — permet déjà du matching basique
- **Profil "Découverte complète" dès J+7** — permet le matching équilibre standard
- **Profil "Mature" dès J+21** — permet le matching équilibre premium + entretien IA Le Miroir
- **Profil "Vivant" dès J+22** (Acte 3 entamé) — top recommandation, top match priority

## 3. ACTE 1 — La Découverte (Jours 1-7)

### Format
- **5-7 min par session**, une session par jour
- **Voix narratrice** (style Petit Bambou : calme, posée, bienveillante)
- **Thématique colorée** : chaque jour a sa couleur, son atmosphère, son focus
- **Pas de bouton "passer"** sans avoir vu la session — mais on peut décaler à demain
- **Pas de score, pas de jugement** affiché à l'utilisateur

### Les 7 chapitres

#### Jour 1 — 🔵 Le rythme (Bleu)
*"Aujourd'hui, on va découvrir comment tu travailles. Pas ce que tu fais — comment tu fonctionnes. Prends ton temps. Il n'y a pas de bonne réponse."*

Mécanique : 4-6 micro-scénarios + 2-3 questions ouvertes courtes.
- Rapide vs réfléchi
- Sprints vs marathon
- Pic d'énergie : matin / soir / variable
- Tolérance à l'imprévu

#### Jour 2 — 🟢 La communication (Vert)
- Directe / diplomate / nuancée
- Écrit / oral / visuel
- Sync (réunions) / async (Slack/email)
- Combien de feedback : continu / hebdo / sur demande

#### Jour 3 — 🟡 L'autonomie (Jaune ambre)
- Encadré / indépendant / collaboratif
- Décide seul / consulte / propose
- A besoin de contexte beaucoup / juste / minimal

#### Jour 4 — 🟣 Les limites (Violet)
*"Aujourd'hui on parle de ce que tu ne veux pas. C'est aussi important que ce que tu veux."*
- Horaires non-négociables
- Types de mission refusés
- Environnements à fuir
- Comportements rédhibitoires

**Cette session est exceptionnellement importante.** C'est elle qui fait que les missions proposées respectent le candidat.

#### Jour 5 — 🟠 L'environnement (Orange)
- Startup chaos / scale-up structuré / corporate process / hybride
- Présentiel / hybride / full remote
- Petite équipe (<5) / moyenne (5-20) / grande (>20)
- Industrie : généraliste / passion / éthique sectorielle

#### Jour 6 — 🔴 La pression (Rouge)
- Stress positif (j'en ai besoin) / neutre / négatif (je fuis)
- Réaction sous pression : sang-froid / créatif / paralysie / fuite
- Gestion d'urgences : routine / exceptionnel acceptable / inacceptable

#### Jour 7 — ⚪ La synthèse (Blanc / lumière)
*"Bravo. Tu as terminé La Découverte. Voici ton portrait."*

Affiche au candidat :
- **3 forces** identifiées (formulées positivement, jamais comme étiquettes)
- **3 préférences fortes** ("Tu fonctionnes mieux quand…")
- **3 limites claires** (assumées, valorisées comme une force)
- **Sa "couleur dominante"** (une couleur de l'arc-en-ciel + une couleur complémentaire)

**Le candidat peut décider de continuer** vers Acte 2 immédiatement ou de revenir plus tard. **L'app encourage doucement à continuer.**

### UX clés
- **Mode focus** — l'app affiche un mode "Voyage" sans notification, sans distraction
- **Audio optionnel** — voix narratrice (ElevenLabs Multilingual v3 FR) + sous-titres + lecture silencieuse possible
- **Ambient sound** — 1 piste sonore par chapitre (7 ambiances), conçues par un sound designer (~5 000 € brief, voir [12](12-DESIGN-PREMIUM-METHODOLOGIE.md))
- **Animations Rive** — illustrations interactives par chapitre, transitions colorées entre jours, 60 fps GPU (~15 000 € studio Rive, voir [12](12-DESIGN-PREMIUM-METHODOLOGIE.md))
- **Micro-feedbacks audio** — tap, swipe, carte débloquée — discrets mais signature
- **Sauvegarde automatique** — on peut interrompre et reprendre
- **Pas de timer visible** — on est dans la posture, pas la performance
- **"Tu peux toujours revenir modifier"** — affiché systématiquement après chaque réponse
- **`prefers-reduced-motion`** respecté — animations désactivables (WCAG 2.2 AA)

## 4. ACTE 2 — Les Mises en Situation (Jours 8-21)

### Format
- **5-8 min par session**, une session par jour
- **Mises en situation réalistes**, métiers, branchantes (chaque choix ouvre une suite différente)
- **Pas de "bonne réponse"** — le but est de capturer le **style**, pas la justesse
- À la fin de chaque scénario, **mini-réflexion** : "Si c'était à refaire, tu changerais quoi ?"

### Catégories de scénarios

#### Décision sous incertitude (3 scénarios)
*Ex : Un client te demande de commencer une mission demain, ton planning est plein. Que fais-tu ?*
- Mesure : risque, négociation, gestion de soi

#### Conflit & feedback (3 scénarios)
*Ex : Un collègue critique publiquement ton travail. Tu réponds comment ?*
- Mesure : régulation émotionnelle, communication directe vs diplomate

#### Négociation (2 scénarios)
*Ex : On te propose une mission 30 % au-dessus de ton tarif mais en environnement chaotique. Tu acceptes ?*
- Mesure : tolérance compromis, alignement valeurs vs argent

#### Pression & urgence (2 scénarios)
*Ex : Tu reçois 4 demandes urgentes simultanées. Tu fais comment ?*
- Mesure : priorisation, calme sous pression

#### Éthique professionnelle (2 scénarios)
*Ex : Tu remarques une erreur que personne d'autre n'a vue, mais corriger te coûte 2h non facturées. Tu fais quoi ?*
- Mesure : intégrité, sens du collectif

#### Apprentissage & échec (2 scénarios)
*Ex : Tu rates un livrable important. Tu apprends quoi de cette expérience ? Comment tu réagis vis-à-vis du client ?*
- Mesure : posture d'apprentissage, transparence, résilience

### Mécanique de branchement
Chaque scénario a **3-4 choix de réponse**, chacun ouvrant une **suite différente** (2-3 niveaux de profondeur). Le candidat ne voit pas son score — il vit l'expérience.

```
Scénario départ
├── Réponse A → Suite A1 → Suite A1.1 / A1.2
├── Réponse B → Suite B1 → Suite B1.1 / B1.2
├── Réponse C → Suite C1 → Suite C1.1
└── Réponse D → Suite D1 → Suite D1.1
```

### À la fin de l'Acte 2 (J+21)
*"Tu viens de vivre 13 situations. Voici ce qu'elles disent de toi."*

Affiche au candidat :
- **Son profil de décision** ("Tu décides rapidement, mais tu consultes systématiquement avant les enjeux majeurs.")
- **Son profil de conflit** ("Tu privilégies la diplomatie, mais tu deviens direct quand la valeur de ton travail est remise en cause.")
- **Son profil de valeurs** (top 5 valeurs perçues)
- **Sa zone de croissance** (formulée positivement)

## 5. ACTE 3 — Les Échos (Jours 22+, en continu)

### Format
- **2-3 min par jour**, optionnel
- Une question, un mini-scénario, ou un retour sur une mission récente
- **Streaks doux** : si tu en fais 5 dans la semaine, tu vois une jolie animation calme — pas un confetti popup
- **Le but :** garder le profil **vivant** dans le temps

### Types d'Échos

#### L'Écho réflexion (40 %)
Une question contextuelle :
- *"Cette semaine, qu'est-ce qui t'a énergisé ?"*
- *"Y a-t-il un type de mission que tu ne ferais plus aujourd'hui ?"*
- *"Comment tu te situes avec ton tarif actuel ?"*

#### L'Écho mini-scénario (30 %)
Une mise en situation très courte, 1 question, 1 réponse.

#### L'Écho retour mission (20 %)
Si tu viens de finir une mission JOB'S ETHIC :
- Comment c'était vraiment ?
- Tu referais avec cette équipe ?
- Ton profil a-t-il évolué ?

#### L'Écho mise à jour (10 %)
*"Au fait, est-ce que tes préférences ont bougé sur le rythme ? On peut ajuster."*

### Pourquoi c'est génial
- **Anti-triche définitif** : le profil mûrit sur 6 mois, 1 an, 3 ans. Personne ne joue un rôle aussi longtemps.
- **Engagement non-intrusif** : 2 min/jour, jamais obligatoire
- **Donnée vivante** : on capte les changements de vie pro réels
- **Switching cost** : "j'ai 18 mois d'Échos là-dedans, je ne vais pas tout recommencer"

### Tonalité
**Calme et FR.** Pas de "Tu es au TOP! 🔥🔥🔥". Plutôt :
- *"Joli. Trois Échos cette semaine. Ton profil prend de la profondeur."*
- *"Ça t'a parlé aujourd'hui ?"*

## 6. La gamification : règles du jeu

### Ce qu'on FAIT
- **Couleurs thématiques** (bleu, vert, jaune, violet, orange, rouge, blanc) — visuel, mémorable, premium
- **Voyage narratif** (Découverte → Mises en Situation → Échos) — sens progressif
- **Cartes profil** — chaque insight devient une "carte" du candidat (collectionnable au sens noble)
- **Streaks doux** — visualisation calme, jamais punitive
- **Niveaux de profil** — "Naissant / Découvert / Mature / Vivant" (basés sur la profondeur réelle, pas sur l'effort)
- **Le Miroir** — moments où le candidat voit son propre portrait, fierté de soi-connaissance

### Ce qu'on REFUSE
- ❌ Points, scores visibles au candidat
- ❌ Badges "compétitifs"
- ❌ Leaderboard
- ❌ Confettis, popups, sons "TADA"
- ❌ Notifications agressives ("Tu vas perdre ton streak!")
- ❌ Comparaisons sociales ("Tu fais mieux que 73 % des utilisateurs")
- ❌ Pression au binge

### Le ton
Si l'app était une voix : **un coach calme, intelligent, qui ne juge pas, qui aide à se découvrir.**

Pas :
- ❌ un prof
- ❌ un gourou
- ❌ un copain trop familier
- ❌ un service client corporate

## 7. Ce qu'on capture précisément (et ce qu'on ne capture jamais)

### Dimensions capturées (le profil profond contient ~30 dimensions)

| Famille | Dimensions |
|---|---|
| **Rythme** | Vitesse, sprint vs marathon, pics d'énergie, tolérance interruption |
| **Communication** | Style (direct/diplomate), canal préféré, fréquence feedback, gestion conflit |
| **Autonomie** | Niveau, besoin de contexte, mode décision (seul/consultatif) |
| **Limites** | Horaires, types mission refusés, environnements à fuir, comportements rédhibitoires |
| **Environnement** | Type structure, taille équipe, remote/présentiel, industries |
| **Pression** | Tolérance, type stress, réaction sous urgence |
| **Décision** | Vitesse, méthode, gestion incertitude |
| **Valeurs** | Top 5 perçues |
| **Croissance** | Zone de progression, posture apprentissage, gestion échec |
| **Collaboration** | Style, rôle préféré (leader / suiveur / partenaire), gestion groupe |

### Dimensions JAMAIS capturées
- ❌ Origine ethnique, perçue ou inférée
- ❌ Genre (au-delà de l'usage neutre des pronoms)
- ❌ Religion
- ❌ Orientation sexuelle
- ❌ Situation familiale détaillée
- ❌ Apparence physique
- ❌ Adresse précise
- ❌ Opinion politique
- ❌ Santé / handicap
- ❌ Émotion en temps réel (interdit par AI Act en contexte pro)

**Tout ce qu'on capture est expliqué au candidat avec un "pourquoi on demande ça".**

## 9. Garde-fous légaux & psychométriques

### Validité psychométrique
- Le profil est construit en collaboration avec un **psychologue du travail certifié** (à recruter Phase 1)
- Le contenu est **inspiré du Big Five revisité** + littérature sur les mises en situation
- **Validation post-hoc** : on mesure la prédictivité du profil sur la durée des missions et l'absence d'échecs
- **Aucune typologie réductrice** (pas de "tu es INTJ", pas de "tu es Rouge/Bleu/Jaune") — on parle en dimensions continues

### Conformité légale
- Toutes les questions passent un **filtre de discrimination** : pas de question qui infère origine, religion, situation familiale, etc.
- **Anonymisation partielle** des profils côté recruteur en Day-1 (prénom + initiale + photo générique optionnelle)
- **DPIA réalisée et publique** (analyse d'impact RGPD)
- **Audit de biais trimestriel** réalisé par un tiers + publié
- **Right to explanation** intégré natif : chaque dimension du profil est expliquée au candidat
- **Right to contest** intégré natif : le candidat peut contester l'interprétation et obtenir révision

Voir [10-ETHIQUE-RGPD-AI-ACT.md](10-ETHIQUE-RGPD-AI-ACT.md) pour le détail.

## 10. Pourquoi Le Voyage est un moat (vrai)

| Moat | Pourquoi c'est défendable |
|---|---|
| **Donnée comportementale propriétaire** | 6+ semaines de micro-data × N utilisateurs = corpus que personne ne peut répliquer en moins de 2 ans |
| **Switching cost asymétrique** | Le candidat ne perdra pas 6 semaines pour aller voir ailleurs (sauf si on l'oblige) |
| **Cluely-resistant by design** | Impossible de truquer 30+ sessions sur 6 semaines |
| **Brand expérientiel** | "Le Voyage" devient un objet culturel — comme Petit Bambou pour la méditation |
| **Compounding** | Plus on a d'utilisateurs et d'années, plus le matching est précis |
| **Loi RGPD pro-nous** | Portable mais cher à recopier ailleurs (le format, l'expérience, l'interprétation sont nous) |
| **Standard d'exécution top 1 %** | Le combo Rive + sound design + voix premium + tempo calme + accessibilité native est extrêmement dur à rattraper sans 18 mois de discipline. Voir [12](12-DESIGN-PREMIUM-METHODOLOGIE.md). |

**Et surtout :** la concurrence ne peut pas copier *Le Voyage* en 3 mois. Même AssessFirst, qui a 20 ans de psychométrie, ne peut pas pivoter du test 30 min vers le voyage 6 semaines sans casser son modèle économique enterprise. Et même s'ils le pouvaient, **ils n'ont pas la culture design pour le faire au niveau premium attendu**.

## 11. Roadmap Le Voyage

### Phase 0 — Validation (Mois 1-2)
- Co-design avec 5 freelances pilotes
- Co-design avec 1 psychologue du travail
- Prototypage Figma + voice over (Wavenet ou ElevenLabs)
- 1 chapitre Découverte fonctionnel

### Phase 1 — MVP (Mois 3-5)
- **Acte 1 complet** (7 chapitres)
- 5 scénarios Acte 2
- Tableau de bord candidat
- Export profil

### Phase 2 — V1 (Mois 6-8)
- **Acte 2 complet** (13 scénarios)
- Début Acte 3 (Échos quotidiens)
- Le Miroir intégré entretien IA
- Validation psychométrique mesurée

### Phase 3 — V2 (Mois 9-12)
- Acte 3 complet et raffiné
- Premières adaptations métiers (VA / customer success / ops / créatif)
- Audit de biais publié

### Phase 4 — V3 (12-18 mois)
- Profil multilingue (anglais, espagnol)
- API "Profil portable" pour consommation externe
- Premiers métiers tech / créatif (élargissement)
